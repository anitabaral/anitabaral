Research

Enhancing ALPR Systems

The purpose of the project was to assess the use of SRGAN (Super-resolution Generative Adversarial Network) to enhance the performance of the ALPR (Automatic License Plate Recognition) systems. The research done for the project has been published in Springer.

RARA: Road-marking Automation

Reprogrammable Automatic Road Aligner (RARA) was built to mark roads, courts, and parking using Thermoplastic Paint, via a reprogrammable interface. Automating the manual work of road-marking using a rover, run via an Android app.
